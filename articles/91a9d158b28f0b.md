# はじめに
カラム周辺のコマンドをすぐ忘てしまうので、まとめてみました

## カラム追加
```
rails g migration Addカラム名Toテーブル名 カラム名:型名
```
- 具体的な書き方

「books」テーブルに文字列型の「title」というカラムを追加したい場合は、以下のようになります。

```
rails g migration AddTitleToBooks title:string
rails db:migrate
```
